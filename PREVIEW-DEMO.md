## Color Coding Legend

```
🔵 Blue Badge = Primary Color (Title slides)
🟠 Orange Badge = Accent Color (Content slides)
🎨 Theme Header = Shows AI-selected theme
📊 Preview Panel = Shows slide structure
```

## Interactive States Summary

| State | Preview Button | Generate Button | Preview Panel |
|-------|---------------|-----------------|---------------|
| Initial | Enabled | Enabled | Empty placeholder |
| Previewing | Disabled (⏳) | Enabled | Loading spinner |
| Preview Ready | Enabled (✅) | Enabled | Shows slides |
| Generating | Enabled | Disabled (⏳) | Shows slides |
| Complete | Enabled (✅) | Enabled (✅) | Shows slides |
| Error | Enabled (❌) | Enabled | Empty |
