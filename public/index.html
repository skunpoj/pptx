Anthropic Claude API

To get your API key:
  1. Go to console.anthropic.com
  2. Sign up or log in
  3. Navigate to "API Keys"
  4. Create and copy your key

OpenAI API

To get your API key:
  1. Go to platform.openai.com
  2. Sign up or log in
  3. Navigate to "API Keys"
  4. Create and copy your key

Google Gemini API

To get your API key:
  1. Go to Google AI Studio
  2. Sign up or log in
  3. Click "Create API Key"
  4. Copy your key

OpenRouter API

To get your API key:
  1. Go to openrouter.ai
  2. Sign up or log in
  3. Navigate to "Keys"
  4. Create and copy your key
